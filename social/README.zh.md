# 自然社媒 — ECHO

**一个常态化的自然社媒存在 —— 永不内置发帖、互动或私信自动化。**

[English](README.md) | 简体中文

> 这是 **[Aaron 营销技能库](../docs/README.zh.md)** 中的一个学科 —— 全库 120 个技能、七个学科、共享一套契约。要看整套体系、四层地图与安装步骤，请从[主 README](../docs/README.zh.md) 开始。

自然社媒是一个常态 **L2 · 频道** —— 自有的社媒存在，在公开场合、实时地表达品牌叙事。十六个技能跑通 **ECHO** 循环 —— **E**xplore 探明该在哪、该怎么发声，**C**raft 打磨平台原生素材，**H**ost 运营社区（回复、社交销售、危机），再 **O**bserve 观测提及、声量与暗社交归因。频道名册由 [`channel-registry`](../protocol/channel-registry/SKILL.md) 持有；质量门是 [`social-quality-auditor`](host/social-quality-auditor/SKILL.md)。**本学科不内置任何发帖、互动或私信自动化** —— 它做规划、起草、度量；由人来按下发布。

## 循环 — Explore → Craft → Host → Observe

- **Explore（探明）** —— 挣得你的存在：频道组合与各频道角色、品牌声音与人设、各平台规范与红线，以及一段非推广的预热爬坡。
- **Craft（打磨）** —— 造平台原生素材：受产能约束的内容日历、帖子/长推/轮播、短视频脚本，以及一份 advocacy 项目规格。
- **Host（运营）** —— 运营社区：素材/项目质量门、回复与提及分诊、创始人牵头的社交销售，以及预先起草的危机分级阶梯。
- **Observe（观测）** —— 读信号：提及与情绪脉搏、相对竞品的声量、暗社交归因，以及一个周期稳定、回写记忆的度量闭环。

## 质量门 — ECHO

[ECHO](../references/echo-benchmark.md) 从四个维度评估自然社媒 —— **E**mbeddedness 嵌入度 · **C**raft 手艺 · **H**osting 运营 · **O**bservability 可观测性（40 个稳定 ID）。每次运行**要么**选素材门、**要么**选一个 program-maturity 档 —— 素材构念与运营构念绝不混用。否决项 `E1`/`C1`/`C2`/`H1`/`H2`/`O1`（须带框架名限定 —— `O1`/`O2` 与 ROAS 在文字上撞号）会硬顶或阻断。门为 [`social-quality-auditor`](host/social-quality-auditor/SKILL.md) → SHIP/FIX/BLOCK；共享机制见 [auditor-runbook.md](../references/auditor-runbook.md)。

## 快速开始

```text
/aaron-marketing:social              # 从你的输入推断 ECHO 阶段
/aaron-marketing:social --phase explore | craft | host | observe
```

```text
/aaron-marketing:social boost this   # 路由到 content-amplifier（内容再利用）
```

每个技能都在 **Tier 1** 用你粘贴的数据即可运行，无需任何 API key。

## 推荐场景

| 你的处境 | 从这里开始 | 得到什么 |
|---|---|---|
| 从零起一个频道 | `/aaron-marketing:social --phase explore` | 频道组合、声音档案、各平台规范、预热爬坡 |
| 要一个月的排期又不想过量发帖 | `--phase craft` → `social-calendar-builder` | 受产能约束的内容日历 + 平台原生初稿 |
| 一条帖子爆了（好或坏） | `--phase host` → `crisis-response-planner` / `engagement-inbox-manager` | 缓冲声明 + 升级阶梯，或回复分诊分级 |
| 我们相对竞品的声量如何？ | `--phase observe` → `share-of-voice-tracker` | 周期稳定分母上的 SOV |
| 把一份素材变成一周的内容 | `content-amplifier`（"boost this"） | 跨格式的再利用变体 |
